Test Coverage & Warnings

In this section, I learnt how to test add test coverage using istanbul. Istanbul is a JavaScript library for creating test coverage and is installed via the command npm install --save-dev nyc.

Cofiguring istanbul

Here, we create a script in our package.json file. In the script we pass a reporter and the script istanbul will run to generate coverage reports for. 

  1. "scripts": {
  2.     "test": "cross-env NODE_ENV=development mocha 'test/**/*.spec.js' --exit",
  3.     "test-app": "cross-env NODE_ENV=development mocha 'test/specs/app.spec.js' --exit",
  4.     "coverage": "cross-env NODE_ENV=development nyc --reporter=text npm test"
  5.  }
Run Coverage

To run the coverage report above, enter the command: 

  1. npm run coverage
